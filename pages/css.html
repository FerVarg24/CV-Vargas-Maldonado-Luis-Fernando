¿Qué es CSS?
CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir el aspecto y el formato de un documento HTML, XML u otro tipo de documento estructurado. Se utiliza para definir la presentación visual de un sitio web, incluyendo aspectos como el color, la fuente, el tamaño, el diseño y la disposición de los elementos.






Ventajas de CSS
  • Presentación del contenido:CSS facilita la difusión de contenido en distintos formatos basado en parmaetros nominales.
    Es decir, preferencias del usuario,del navegador,la localización geográfica y el tipo de dispositivo desde el cual se accede al contenido.

  • Lenguaje Sencillo:Es un lenguaje con un tiempo de aprendizaje relativamente corto para realizar buenos trabajos.

  • Archivos Pequeños.

  • Flexibilidad:Podemos cambiar en cualquier momento alguna parte o la totalidad del diseño de nuestras páginas con sólo modificar nuestra hoja de estilo, sin que ello suponga modificar el contenido.

  • Unificación de Diseño:Mantener una misma apariencia en un sitio web es tedioso por lo que enlazar a cada una de nuestras páginas nuestras hojas de estilo,agiliza el proceso.






¿Por que se creó CSS?
CSS fue creado para resolver un problema fundamental en la creación de sitios web: separar el contenido del diseño y la presentación visual. En los primeros días de la web, los sitios web se diseñaban utilizando tablas HTML y atributos de presentación como el tamaño de la fuente y el color de fondo se incluían directamente en el código HTML. Esto hacía que el código HTML fuera más complicado y difícil de mantener.

Además, cualquier cambio en el diseño de un sitio web requería modificar manualmente cada página individual, lo que hacía que la actualización de los sitios web fuera un proceso lento y tedioso. CSS fue diseñado para abordar estos problemas al permitir que el diseño y la presentación visual de un sitio web se definieran en un archivo separado de la estructura del documento HTML.




¿Cuando se creo CSS?
En 1994 se comenzó el desarrollo de CSS por parte de el World Wide Web Consortium debido a que a Tim Berners-Lee le parecía necesario separar la presentación visual de la estructura con la estructura fundamental del documento HTML

Dos años después se publicó el primer borrador de la especificación CSS.Posteriormente, en 1988 se pubnlicó la primera version oficial bajo el nombre de CSS1

En años posteriores se lanzaron las versiones CSS2 y CSS3 que incluían nuevas características haciendolas capaces de realizar diseños más complejos

En la acutalidad,CSS sigue siendo desarrollado por el World Wide Web Consortium y se actualiza constantemente para adaptarse al avance de las tecnologíass




Relacion de CSS con HTML
CSS funciona separadamente del HTML, lo que permite a los desarrolladores de sitios web separar el contenido del diseño y el formato. Esto hace que el desarrollo de sitios web sea más eficiente, ya que los cambios de diseño pueden realizarse fácilmente a través de una hoja de estilo CSS centralizada, en lugar de tener que modificar cada página individualmente.

CSS se aplica a través de reglas que se definen en una hoja de estilo, que se puede enlazar con un documento HTML utilizando una etiqueta <link> en el encabezado del documento o directamente en el atributo "style" de un elemento HTML específico. Las reglas CSS pueden aplicarse a elementos individuales, como un encabezado o un párrafo, o a grupos de elementos que comparten un atributo común, como todos los enlaces o todos los párrafos en una página.
