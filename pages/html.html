¿Qué es HTML?
“HyperText Markup Language”
HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas web sin embargo no se refiere al diseño gráfico de la web si no al orden que llevan los elementos dentro de la página. Esto lo hace por medio de las marcas de hipertexto conocidas como “tags” o etiquetas
Las etiquetas le dan un orden a todo el contenido,por ejemplo:
Si realizáramos un ensayo en una página web las etiquetas nos ayudarían a delimitar el espacio que ocupa cada una de las partes del ensayo,es decir,una etiqueta para el título,otra para la introducción y así sucesivamente.

Las etiquetas se componen de:
  • El carácter “menor que” <
  • Una palabra o carácter que determina qué etiqueta se está escribiendo
  • Cualquier número de atributos HTML que se quiera usar, escritos de la forma nombre =”valor”
  • El carácter “mayor que” >
  • Una diagonal que indica el cierre de la etiqueta





Estructura Basica Html
Al inicio del código HTML se encuentra la declaración <!DOCTYPE html> la cual le informa al navegador la verson de HTML con la que se hixo el documento y como procesarla

Despues encontramos la etiqueta <html> la cual engloba todo el contenido de la pagina web
Seguida de esta se encuentra el elemento head y es aqui donde se utilizan etiquetas que indican cómo debe comportarse el navegador o los motores de búsqueda con la página web.

Dentro de esta etiqueta escribiremos el titulo de la pagina web haciendo uso de la eiqueta <title>.
La siguiente seccion corresponde al <body> que es donde escribimos toda nuestra página web, desde los títulos, párrafos, formularios, imágenes, videos y definimos por medio de etiquetas la tipografía, el color de la letra,etc.




¿Por que se creó HTML?
HTML (HyperText Markup Language) se creó para permitir la creación de páginas web y para hacer posible que los usuarios pudieran acceder y ver información en línea de manera fácil y consistente.
Fue creado por Tim Berners-Lee en 1990,quien quería crear una manera de compartir información de manera electrónica, pero no había una forma fácil de hacerlo. Por lo tanto, Berners-Lee desarrolló un sistema basado en el concepto de hipertexto, que permitía a los usuarios navegar y acceder a diferentes tipos de información en línea.




¿Como se creó HTML?
Para hacer realidad esta visión, Berners-Lee desarrolló el concepto de hipertexto, que permite a los usuarios navegar y acceder a diferentes tipos de información en línea. El hipertexto utiliza enlaces que conectan diferentes documentos, lo que permite a los usuarios acceder a información relacionada de manera fácil y eficiente.

Para implementar el hipertexto, Berners-Lee creó el primer servidor web y el primer navegador web, que permitía a los usuarios acceder a documentos en línea a través de enlaces. Estos documentos se escribieron en un lenguaje llamado HTML, que Berners-Lee creó para dar formato a los documentos y hacer que fueran legibles para el navegador web.

HTML utiliza etiquetas de marcado para definir diferentes elementos en una página web, como encabezados, párrafos, imágenes y enlaces. Estas etiquetas permiten que el navegador web muestre el contenido de una página de manera estructurada y organizada. A lo largo de los años, HTML ha evolucionado y se han agregado nuevas características y funcionalidades. La última versión de HTML es HTML5, que se lanzó en 2014 y es ampliamente utilizada en la creación de páginas web modernas y dinámicas.




¿Cuando se creo HTML?
El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de "hipertexto" para compartir documentos.

En 1995, el organismo IETF organiza un grupo de trabajo de HTML y consigue publicar, el 22 de septiembre de ese mismo año, el estándar HTML 2.0. A pesar de su nombre, HTML 2.0 es el primer estándar oficial de HTML.

Desde 1996, la W3C (World Wide Web Consortium) quedó acargo de los estandares de HTML.En 1997 se publicó la version 3.2 la cual fue la primera recomendación de HTML publicada por el W3C.

HTML 4.0 se publicó el 24 de Abril de 1998 Entre sus novedades más destacadas se encuentran las hojas de estilos CSS y la posibilidad de incluir pequeños programas o scripts en las páginas web

Desde la publicación de HTML 4.01, la actividad de estandarización de HTML se detuvo y el W3C se centró en el desarrollo del estándar XHTML. Por este motivo, en el año 2004, las empresas Apple, Mozilla y Opera mostraron su preocupación por la falta de interés del W3C en HTML y decidieron organizarse en una nueva asociación llamada WHATWG (Web Hypertext Application Technology Working Group) la cual se centra en el futuro estándar HTML 5
